# Проект: my-first-project

## Что это за проект

Персональный веб-проект, созданный в учебных целях. Включает визитку специалиста и лендинг услуги — оба файла представляют собой самодостаточные HTML-страницы без внешних зависимостей. Для локального просмотра используется встроенный PowerShell-сервер.

---

## Проблема

Начинающим специалистам и фрилансерам сложно заявить о себе в интернете: нет ни готового сайта, ни инструментов, чтобы его сделать. А клиентам трудно понять, чем именно может помочь консультант, пока им не объяснят это просто и наглядно. Этот проект решает обе задачи: даёт специалисту личную страницу-визитку и убедительный лендинг для продвижения услуги.

---

## Для кого

- **Визитка** — для Елены Десятовой, вайбкодера. Личная страница-карточка с контактами и ссылками на соцсети.
- **Лендинг** — для финансового консультанта. Целевая аудитория — люди, которые хотят начать накапливать и инвестировать, но не знают с чего начать.

## Основные функции

**Визитка (`vizitka.html`):**
- Отображение имени, специализации и короткого описания
- Иконки соцсетей: Telegram, GitHub, X (Twitter)
- Кнопка-ссылка на Telegram для связи
- Загрузка фото через клик по аватару (локально, без сервера)
- Анимация появления карточки и частицы на фоне

**Лендинг (`landing.html`):**
- Hero-секция с заголовком, подзаголовком и статистикой (+40%, 3 мес., 100%)
- Три шага к финансовой свободе — карточки с описанием услуги
- Секция «Для кого» с целевой аудиторией и прогресс-барами
- Блок «Почему это выгодно» с преимуществами
- Финальный CTA с кнопкой перехода в Telegram
- Плавное появление секций при скролле

## Чего НЕ будет (пока)

- Бэкенда и базы данных — всё статично, без серверной логики
- Формы с реальной отправкой — кнопки ведут в Telegram
- Авторизации и личного кабинета
- CMS или панели управления контентом
- Адаптивной навигации / мобильного меню
- Мультиязычности
- SEO-оптимизации и метатегов для соцсетей (og:image и т.п.)
- Подключения к платёжным системам

## Стиль дизайна

Оба файла выполнены в **тёмном минималистичном стиле** с анимациями:

- **Тёмный фон** — почти чёрный (`#0d0907` у визитки, `#05080f` у лендинга)
- **Акцентные цвета**: тёплый оранжево-коричневый (визитка) и бирюзово-синий с золотом (лендинг)
- **Частицы** — анимированные точки, плавно поднимающиеся на фоне
- **Градиентные рамки** вокруг карточек
- **Плавные появления** элементов при скролле (`IntersectionObserver`)
- **Единый шрифт** — системный стек (`-apple-system`, `BlinkMacSystemFont`, `Segoe UI`)
- Все стили встроены прямо в HTML (`<style>`), без внешних CSS-файлов
