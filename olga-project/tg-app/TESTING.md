# Руководство по тестированию — Ольга · маникюр (TMA)

## О проекте

Telegram Mini App для записи к мастеру маникюра.
Весь код — один файл: `olga-project/tg-app/index.html`.

**Живой URL:** `https://olga-project.vercel.app`
**Бот:** `@Ma_master_bot`
**GitHub:** `desyatovaelena11/my-projects`, ветка `main`, папка `olga-project/tg-app`

---

## 1. Запуск локально

```bash
# Клонировать репозиторий
git clone https://github.com/desyatovaelena11/my-projects.git
cd my-projects/olga-project/tg-app

# Запустить любой локальный сервер, например:
npx serve .
# или
python -m http.server 8080
```

Открыть в браузере: `http://localhost:8080`

> Без локального сервера (через `file://`) часть браузерных API может не работать.

---

## 2. Тестирование в браузере (без Telegram)

При открытии вне Telegram автоматически появляются **fallback-кнопки** внизу экрана:
- «← Назад» — аналог Telegram BackButton
- Синяя кнопка — аналог Telegram MainButton

Данные пользователя будут пустыми: в приветствии онбординга покажется «Привет!» вместо имени.

---

## 3. Тестирование в Telegram

1. Открыть бота `@Ma_master_bot`
2. Нажать кнопку **Записаться** (Menu Button) внизу чата
3. Mini App откроется на весь экран

Альтернативный способ: написать боту `/start`.

---

## 4. Управление localStorage (сброс состояния)

TMA использует два ключа в localStorage:

| Ключ | Значение | Поведение |
|---|---|---|
| `onboarding_done` | `"1"` | Онбординг не показывается |
| `offer_shown` | `"1"` | Оффер-модал не показывается |

**Как сбросить** (DevTools → Application → Local Storage → выбрать домен → Delete All):

- В браузере: F12 → Application → Local Storage → `http://localhost:8080` → Clear All
- В Telegram Desktop: нажать ⋮ (три точки) в углу Mini App → Inspect → открывается DevTools

**Как сбросить через консоль:**
```js
localStorage.removeItem('onboarding_done');
localStorage.removeItem('offer_shown');
location.reload();
```

## 8. Деплой

Vercel автоматически деплоит при `git push` в ветку `main`.
Root Directory в настройках Vercel: `olga-project/tg-app`.

```bash
git add olga-project/tg-app/index.html
git commit -m "описание изменений"
git push
# Vercel задеплоит за ~1 минуту
```

После деплоя обновить Mini App в Telegram: ⋮ → Reload page.
